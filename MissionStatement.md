# Goals
The features for the final product.
- Acid compliant.
- Can be any datatype stored, not just text. BLOBs.
- Idle Optimization (When CPU idling, compress and defragment)
- Single-File sharing of databases. IE Database can be compacted to a single file for easy sharing, and unpacked by another instance.
- A UI App (Like Oracle's SQL Viewer)
- A scripting Language ontop of the query langauge
- Encryption, Compression, and checksums (with ECC?)
